# Terrarium
   A payload containing soil, plant life and water that will be flown in the rocket.

## Sensors
- Humidity + Temperature (waterproof):
   -[SHT3x-DIS](https://www.adafruit.com/product/4099)
   -[datasheet](https://cdn-shop.adafruit.com/product-files/2857/Sensirion_Humidity_SHT3x_Datasheet_digital-767294.pdf)

   -[LSM9DS1](https://www.adafruit.com/product/3387)
   -[datasheet](https://www.st.com/resource/en/datasheet/lsm9ds1.pdf)
- Barometric Pressure
   -[BMP390](https://www.adafruit.com/product/4816)
   -[datasheet](https://www.bosch-sensortec.com/media/boschsensortec/downloads/datasheets/bst-bmp390-ds002.pdf)
